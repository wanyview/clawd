# 燧光知识沙龙 - 前端规范

> 燧光：取"燧人之火，光照万里"之意，象征知识的传播与点亮

## 1. 项目概述

- **项目名称**: 燧光知识沙龙 (SuiLight Salon)
- **类型**: 单页Web应用 (SPA)
- **核心理念**: 知识碰撞产生火花，理念传播点亮智慧
- **风格**: 温暖、智慧、碰撞、启发

## 2. 设计规范

### 色彩系统

| 用途 | 颜色 | Hex |
|------|------|-----|
| 主色(燧光橙) | 温暖火光 | #FF6B35 |
| 辅色(知识蓝) | 智慧深邃 | #1E3A5F |
| 强调色(金色) | 灵感火花 | #FFD700 |
| 背景色 | 温暖黑夜 | #0D1117 |
| 文字色 | 柔光白 | #E6EDF3 |
| 次要文字 | 柔和灰 | #8B949E |

### 字体

- 标题: "ZCOOL XiaoWei", serif (中文)
- 正文: "Noto Sans SC", sans-serif
- 代码: "JetBrains Mono", monospace

### 视觉效果

- 粒子背景效果 (模拟火花)
- 卡片悬浮发光效果
- 渐变边框动画
- 打字机效果展示引用

## 4. 交互功能

- 粒子背景动画
- 卡片悬停发光效果
- 标签云点击筛选
- 知识胶囊卡片展示
- 模拟打字机效果

## 5. 技术栈

- 纯 HTML/CSS/JavaScript
- 无需构建工具
- 响应式设计 (移动端适配)
