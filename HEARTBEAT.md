## 汇报机制（立即执行）

### 每日三次主动汇报
- **08:00** 晨间汇报：今日计划+服务状态
- **12:00** 午间汇报：上午进度+问题
- **18:00** 晚间汇报：全天总结+明日计划

### 汇报内容模板
```
📊 进展汇报 - [时间]
1. 今日P0任务进度
2. 关键里程碑状态
3. 遇到的问题
4. 需要决策的事项
```

## 自动触发汇报
- 服务异常时 → 立即警报
- 任务完成时 → 即时通知
- 里程碑达成 → 正式汇报
- 遇到阻塞 → 寻求帮助

## 心跳检查间隔
每30分钟检查一次并更新状态
