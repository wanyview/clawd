## 四、写作规范

### 论文结构

```
1. Abstract (摘要)
2. Introduction (引言)
3. Related Work (相关工作)
4. Methodology (方法)
5. Experiments (实验)
6. Results (结果)
7. Discussion (讨论)
8. Conclusion (结论)
9. References (参考文献)
```

### 摘要模板 (300字)

```
[背景]
[问题]
[方法]
[结果]
[贡献]
```
