# 附中M2胶囊接口方案

**版本**: V2.0  
**状态**: 待定稿  
**日期**: 2026-03-03

---

## 1. 概述

M2方案旨在为附中知识矩阵系统提供标准化的胶囊API接口，实现与现有Edu-Matrix系统的无缝对接。

### 1.1 目标
- 提供RESTful API接口
- 支持胶囊的创建、查询、搜索、推荐
- 兼容现有Matrix-BNUHS-v2系统

### 1.2 服务地址
```
http://localhost:8005
```

---

## 2. 核心API接口

### 2.1 胶囊管理

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /capsules | 创建新胶囊 |
| GET | /capsules | 列表查询 |
| GET | /capsules/{id} | 获取详情 |
| PUT | /capsules/{id} | 更新胶囊 |

### 2.2 搜索接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /capsules/semantic-search | 语义搜索 |
| GET | /capsules/{id}/related | 相关胶囊 |

### 2.3 拉取接口（供外部系统调用）

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/v1/capsule/pull | 拉取胶囊 |
| GET | /api/v1/capsule/pull/domains | 获取领域列表 |
| GET | /api/v1/capsule/pull/tags | 获取标签列表 |

### 2.4 推荐接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /recommend | 个性化推荐 |

---

## 3. 接口详情

### 3.1 创建胶囊

```bash
POST /capsules
Content-Type: application/json

{
  "title": "胶囊标题",
  "content": "胶囊内容",
  "domain": "教育",
  "tags": ["知识沙龙", "AI"],
  "author": "Kai",
  "level": "standard"
}
```

**返回**:
```json
{
  "id": "sha256_hash",
  "status": "draft",
  "created_at": "2026-03-03T..."
}
```

### 3.2 语义搜索

```bash
POST /capsules/semantic-search
Content-Type: application/json

{
  "query": "什么是知识胶囊？",
  "top_k": 10,
  "domain": "教育"
}
```

### 3.3 拉取胶囊（外部接口）

```bash
POST /api/v1/capsule/pull
Content-Type: application/json

{
  "domain": "教育",
  "tags": ["知识沙龙"],
  "limit": 50,
  "since_id": "optional_cursor"
}
```

## 4. 对接Matrix-BNUHS-v2

### 4.1 集成方式

在Matrix系统中调用胶囊服务:

```typescript
// Matrix-BNUHS-v2/services/capsule.ts
const CAPSULE_API = 'http://localhost:8005';

export async function searchCapsules(query: string) {
  const res = await fetch(`${CAPSULE_API}/capsules/semantic-search`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ query, top_k: 10 })
  });
  return res.json();
}
```

### 4.2 认证方式

- 内部调用: 无认证
- 外部调用: API Key (后续版本)
