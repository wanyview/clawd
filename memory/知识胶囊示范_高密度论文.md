## 🌿 核心胶囊 (密度≥90)

### Capsule 1: 核心理念
```
标题: Knowledge Capsule定义
内容: Knowledge Capsule是一种自我描述的知识单元，
      结合原始内容与AI生成的元数据。
      不同于简单笔记，知识胶囊包含丰富的语义注释，
      支持精确检索和上下文发现。

密度: 95分 | 标签: #核心概念 #AI #知识管理
```

### Capsule 2: 元数据架构
```
标题: 知识胶囊元数据Schema
内容: 每个知识胶囊包含6类元数据:
      1. 内容元数据 (原始文本、字数、时间戳)
      2. 主题元数据 (主题标签、置信度、层级)
      3. 概念元数据 (摘要、关键点、问答)
      4. 实体元数据 (人物、组织、地点、概念)
      5. 关系元数据 (相关胶囊、先验知识)
      6. 重要性元数据 (重要性分数、复用概率)

密度: 92分 | 标签: #数据结构 #Schema #技术架构
```

      Stage 2: AI分析 - LLM主题分类、摘要生成、实体提取
