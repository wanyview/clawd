# ☕ 每日咖啡推荐系统 - 需求文档

*创建时间: 2026-03-04*

---

## 一、需求概述

根据用户每日状态/任务历史，推荐定制咖啡

## 二、推荐模式

### 模式1: 品牌推荐
推荐国内品牌咖啡（蜜雪冰城、瑞幸、库迪等）适合今天的款式

### 模式2: 手冲配方
推荐手冲咖啡配方（豆子、比例、水温等）

## 三、推荐逻辑

### 输入因素
- 昨日任务完成情况
- 当前精力状态
- 天气/季节
- 用户偏好

### 输出
- 咖啡名称
- 推荐理由
- 适合场景

## 四、数据准备

### 咖啡库
- 品牌咖啡库（瑞幸、库迪、蜜雪等）
- 手冲配方库（意式、单品、拿铁等）

### 用户画像
- 口味偏好
- 咖啡因敏感度
- 饮用时间

---

## 五、技术实现

### API设计
- `/api/v1/coffee/recommend` - 获取每日推荐
- `/api/v1/coffee/brands` - 品牌列表
- `/api/v1/coffee/recipes` - 配方列表

### 整合
- 知识胶囊 API
- 用户数据
- 外部咖啡数据
