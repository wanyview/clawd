# 李秘书提醒机制

*制定时间: 2026-03-03 23:42*

---

## 提醒规则

### 1. 任务完成提醒
每次完成任务后，提醒生成新任务

### 2. 定时提醒

| 时间 | 提醒内容 |
|------|----------|
| 08:00 | 晨间汇报 + 今日任务 |
| 12:00 | 午间检查 + 补充任务 |
| 18:00 | 晚间复盘 + 明日任务 |
| 21:00 | 自检反思 + 补充待办 |

### 3. 关键提醒

- 完成任务时 → 立即生成新任务
- 任务池空了 → 立即补充
- 目标未达成 → 立即行动

---

## 提醒话术

### 完成任务时
```
任务已完成！请立即生成下一个任务，保持永动机运转。
```

### 任务池空了
```
任务池已空！请立即补充新任务。
```

### 定时提醒
```
时间到了！请执行[晨间汇报/午间检查/晚间复盘/自检反思]
```

---

## 执行

- Cron定时触发
- 每次任务完成后触发
- 任务池检查时触发
