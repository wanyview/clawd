# 📋 本地大模型方案

> 制定时间: 2026-02-28 21:00

## 一、背景

### 为什么需要本地大模型？

| 因素 | 说明 |
|------|------|
| **数据安全** | 知识胶囊数据不出网 |
| **成本控制** | 减少API调用费用 |
| **响应速度** | 本地推理更快 |
| **定制化** | 针对知识胶囊场景优化 |

| 模型 | 参数 | 场景 | 状态 |

## 三、实施计划

### 3.1 部署步骤

```bash
# 1. 安装 Ollama
brew install ollama

# 2. 拉取模型
ollama pull qwen2.5:7b

# 3. 启动服务
ollama serve

# 4. 测试
curl -X POST http://localhost:11434/api/generate -d '{"model":"qwen2.5:7b","prompt":"你好"}'
```
