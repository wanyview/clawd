# 永动机运转机制

*制定时间: 2026-03-03 23:41*

---

## 核心原则

**永远有在途任务** - 任务完成后立即生成新任务

---

## 运转流程

```
完成一个任务 → 立即生成下一个任务 → 执行 → 验证 → 交付 → 归档 → 再生成新任务
```

---

## 每日循环

| 时间 | 动作 |
|------|------|
| 08:00 | 晨间汇报 + 生成今日任务 |
| 12:00 | 检查进度 + 补充新任务 |
| 18:00 | 复盘 + 生成明日任务 |
| 21:00 | 自检 + 补充待办 |
| 任何时刻 | 完成任务 → 立即生成新任务 |

4. **每月至少1次大迭代**
