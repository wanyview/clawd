## 📊 四维评分模型

### 综合决策公式

```
综合评分 = 科技分 × 0.3 + 学术分 × 0.2 + 国际分 × 0.25 + 货币分 × 0.25

评分标准:
≥80分: 重仓买入
60-80分: 正常持仓
40-60分: 观望
<40分: 卖出
```

### 评分维度详解

| 维度 | 评估内容 | 正面信号 | 负面信号 |
|------|----------|----------|----------|
| **科技** | 核心技术 | 顶会论文增长 | 技术落后 |
| **学术** | 产业化阶段 | 论文爆发期 | 行业下行 |
| **国际** | 地缘政治 | 合作共赢 | 冲突升级 |
| **货币** | 资金流向 | 外资流入 | 资金外流 |

## 🎯 实战案例

### 案例: 宁德时代

```
四维评分:

1. 科技分: 90分
   → 固态电池技术领先
   → 全球市占率第一

2. 学术分: 85分
   → 电池论文持续增长
   → 产业化成熟期

3. 国际分: 60分
   → 中美科技竞争
   → 国产替代加速(正面)

4. 货币分: 70分
   → 新能源赛道受资金青睐
   → 外资配置增加

综合评分: 77.5分 → 持有/加仓
```

### 案例: 某芯片公司

```
四维评分:

1. 科技分: 75分
   → 成熟制程
   → 先进制程待突破

2. 学术分: 80分
   → 论文数量增加
   → 人才培养中

3. 国际分: 40分
   → 受制裁影响大
   → 供应链风险

4. 货币分: 65分
   → 国产替代受政策支持

综合评分: 65分 → 观望
```
