## 三、实施方案

### 3.1 本地大模型调用

```python
import requests

def call_local_llm(prompt: str, model: str = "qwen2.5:7b") -> str:
    response = requests.post(
        "http://localhost:11434/api/generate",
        json={"model": model, "prompt": prompt, "stream": false}
    )
    return response.json()["response"]
```
