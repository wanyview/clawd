## 📚 股票胶囊库

### 🔴 红色层 - 人类理解

#### 1. 龙头股识别胶囊
```
【定义】
龙头股是某一行业中具有代表性的股票，
往往先于行业上涨，带动板块联动。

【特征】
• 市值较大、行业地位高
• 成交量活跃、资金关注
• 先于板块启动、涨幅领先

【判断标准】
• 行业涨幅前三
• 成交量是平时的2倍以上
• 连续3天涨幅>5%
```

#### 2. 趋势交易胶囊
```
【定义】
顺势而为，在上升趋势中买入，在下降趋势中卖出。

【核心指标】
• 均线：MA5>MA10>MA20 做多
• MACD：金叉买入、死叉卖出
• 成交量：涨时放量、跌时缩量

【纪律】
• 止损：-7%必须卖出
• 止盈：+15%可考虑卖出
```

#### 3. 风险控制胶囊
```
【定义】
保护本金、控制回撤是炒股第一要务。

【仓位管理】
• 单只股票不超过30%仓位
• 总仓位不超过80%
• 保留20%现金备用

【止损纪律】
• -5%预警
• -7%止损
• -10%强制平仓
```
