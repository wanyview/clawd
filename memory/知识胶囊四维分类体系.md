## 🎯 应用场景详解

### 🔍 知识发现型胶囊
- **核心**: 帮助用户从已有知识中找到关联
- **触发**: 用户问"是什么"、"有什么区别"
- **典型**: 定义、比较、分类、案例

### 💡 知识发明型胶囊
- **核心**: 触发用户产生新想法
- **触发**: 用户问"如何创新"、"有什么新方法"
- **典型**: 原理、机制、突破、前沿

### 🏭 产业创新型胶囊
- **核心**: 将知识转化为产品/服务
- **触发**: 用户问"如何落地"、"如何商业化"
- **典型**: 技术方案、商业模式、产业链

### 🤖 支撑AI型胶囊
- **核心**: 为AI Agent提供可调用知识
- **触发**: AI需要检索、推理、执行
- **典型**: 数据结构、函数定义、逻辑规则

### 🔄 知识复现型胶囊
- **核心**: 复现历史上的重大发明发现
- **触发**: 用户想学习"如何做到"
- **典型**: 过程还原、要素分解、步骤指南
