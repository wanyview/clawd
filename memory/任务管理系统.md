## 一、任务流转机制

```
立项 → 分配 → 执行 → 检查 → 复盘 → 归档
```

| 阶段 | 负责 | 产出 |
|------|------|------|
| 立项 | 曹操(CEO) | 任务定义、验收标准 |
| 分配 | 各牵头人 | 责任人、时间节点 |
| 执行 | 责任人 | 阶段性产出 |
| 检查 | 赵云(东向) | 进度报告 |
| 复盘 | 曹操 | 经验总结 |
| 归档 | - | 文档沉淀 |

- 复盘经验教训
