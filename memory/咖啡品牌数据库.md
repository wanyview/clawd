## 三、咖啡豆品种知识

### 阿拉比卡 (Arabica)
- **占比**: 全球60-70%
- **特点**: 风味细腻、酸度适中、花果香丰富
- **咖啡因**: 较低 (1.2-1.5%)

### 罗布斯塔 (Robusta)
- **占比**: 全球30-40%
- **特点**: 苦味重、咖啡因高、醇厚度高
- **咖啡因**: 较高 (2.2-2.7%)

### 利比里卡 (Liberica)
- **占比**: <2%
- **特点**: 风味独特、花香浓郁

## 四、推荐算法

| 用户状态 | 推荐 |
|----------|------|
| 精力充沛 + 早晨 | 拿铁类 |
| 精力一般 + 午后 | 美式/浓缩 |
| 疲惫 + 需提神 | 浓缩/冰美式 |
| 寒冷天气 | 热拿铁 |
| 炎热天气 | 冰美式/特调 |
| 甜口味 | 生椰拿铁 |
| 苦口味 | 美式/浓缩 |
